# 🎨 Мамина Студія / Mama's Studio

Безлімітна, дружня до користувача студія створення зображень для мами 💛
Працює на **Ideogram 4 (open-weight)** через **2× NVIDIA T4** на Kaggle, з публічним
доступом через **ngrok**. Інтерфейс — українською, схожий на дружній чат.

> Unlimited, friendly AI image studio. Runs **Ideogram 4 (open-weight)** on
> **2× NVIDIA T4** (Kaggle), exposed publicly via **ngrok**. Ukrainian UI,
> Claude-style friendly chat.

---

## ✨ Можливості

- 🖼️ **4 (і більше) зображення за раз** — обирай найкраще
- 🤖 **Агенти-помічники** — тепло спілкуються українською, перепитують дрібнички,
  самі обирають інструменти (малювати, перетворити фото, збільшити, перевірити)
  й **переглядають готові картинки на дефекти**. Ключ Ollama вводиться прямо в інтерфейсі.
- 💬 **Дружній чат** українською, як у Claude
- ✏️ **«Обведи й зміни»** — обвести пальцем ділянку і замінити її (inpainting)
- 🎭 **Стилі одним дотиком** — фото, олія, акварель, аніме, 3D, казка, олівець, ретро
- 🖌️ **«З мого фото»** — завантаж своє фото і перетвори його (image-to-image)
- 🔍 **Збільшити якість** — зробити зображення більшим і чіткішим (upscale)
- 🎚️ **Повзунок якості** — від «швидко» до «найкраще»
- 🎤 **Голосовий ввід** (українською)
- 🎨 Формати: квадрат / альбом / портрет / банер
- 🌈 Плавні анімації, SVG-іконки, тепла палітра
- 📥 Збереження, історія, варіації
- 🧩 **Демо-режим без GPU** — для перевірки інтерфейсу будь-де

## 🏗️ Архітектура

```
Мама (браузер)
      │  https://….ngrok-free.app
      ▼
   ngrok тунель
      ▼
 FastAPI (server.py)  ──►  фронтенд (index.html / styles.css / app.js)
      │
      ▼
 IdeogramEngine (ideogram_engine.py)
      ├─ GPU 1 (cuda:1): текстовий енкодер Qwen3-VL-8B
      └─ GPU 0 (cuda:0): DiT-трансформер + VAE
```

Велику модель розкладено на **дві відеокарти**, тому 9.3B-модель + 8B-енкодер
вміщаються на 2×16 ГБ. На T4 використовується **nf4**-квантизація (fp8 на T4 не
працює).

## 🚀 Запуск на Kaggle (для мами)

1. **Kaggle → New Notebook**, завантажте `kaggle/MaminaStudia_Kaggle.ipynb`.
2. **Settings → Accelerator → `GPU T4 x2`**, **Internet → On**.
3. Прийміть ліцензію моделі (кнопка «Agree and access repository»):
   <https://huggingface.co/ideogram-ai/ideogram-4-nf4-diffusers>
4. **Add-ons → Secrets** — додайте:
   - `HF_TOKEN` — токен Hugging Face (<https://huggingface.co/settings/tokens>)
   - `NGROK_TOKEN` — токен ngrok (<https://dashboard.ngrok.com>)
   - `OLLAMA_API_KEY` — ключ Ollama Cloud для агентів (<https://ollama.com>) — або введіть його в UI
   - *(необовʼязково)* `IDEOGRAM_API_KEY` — для «магічних» підказок
5. У першій клітинці змініть `REPO_URL` на свій репозиторій (куди запушили проєкт).
6. **Run All** → дочекайтесь рядка `https://….ngrok-free.app` → це посилання дайте мамі. 🎉

> ⚠️ Поки ноутбук відкритий — студія працює. Kaggle дає ~30 год GPU/тиждень безкоштовно.

## 💻 Локальна перевірка (демо-режим, без GPU)

```bash
cd mamina-studia
bash run_local.sh
# відкрий http://localhost:7860
```

У демо-режимі (`MOCK_MODE=1`) сервер генерує красиві зображення-заглушки, щоб
перевірити весь інтерфейс і потік без відеокарти.

## 🔌 API

| Метод | Шлях | Опис |
|------|------|------|
| `GET`  | `/api/health`   | стан двигуна (mock / dual-GPU, розкладка) |
| `POST` | `/api/generate` | текст → N зображень (base64) |
| `POST` | `/api/inpaint`  | «обведи й зміни» (маска + промпт) |
| `POST` | `/api/img2img`  | «з мого фото» (фото + промпт + сила) |
| `POST` | `/api/upscale`  | збільшити та зробити чіткішим |
| `POST` | `/api/chat`     | агенти: розмова → інструменти (Ollama) |
| `POST` | `/api/review`   | перевірка картинок на дефекти (vision) |

### 🤖 Агенти-помічники (Ollama Cloud)

Команда інструментальних агентів на базі **Ollama Cloud** (модель за замовч.
**`gemma4:31b`** — багатомовна + бачить зображення). Вони:
- спілкуються українською, перепитують деталі, коли ідея нечітка;
- самі **обирають інструменти** (всі доступні): `generate`, `img2img`,
  `variations`, `upscale`, `edit`, `review`;
- після генерації **переглядають картинки очима** (агент-критик) і попереджають
  про дефекти, пропонуючи перемалювати з порадами.

**Ключ можна ввести прямо в інтерфейсі:** бічна панель → «Агенти й ключ» →
вставити ключ Ollama → «Перевірити» → «Зберегти». Ключ зберігається лише в
браузері й надсилається у заголовку `X-Ollama-Key`. Як альтернатива — змінна
середовища `OLLAMA_API_KEY` (на Kaggle — секрет). Без ключа застосунок працює
у звичайному режимі «напиши → намалюй». Перемкнути агентів — кнопкою «🤖 Агенти».

Приклад:
```bash
curl -X POST http://localhost:7860/api/generate \
  -H 'Content-Type: application/json' \
  -d '{"prompt":"руде кошеня у віночку з ромашок","count":4}'
```

## ⚙️ Налаштування (змінні середовища)

| Змінна | За замовч. | Опис |
|--------|-----------|------|
| `MOCK_MODE` | `0` | `1` — демо без GPU |
| `DUAL_GPU` | `1` | розкласти модель на 2 GPU |
| `IDEOGRAM_MODEL_REPO` | `ideogram-ai/ideogram-4-nf4-diffusers` | репозиторій ваг |
| `HF_TOKEN` | — | токен Hugging Face (ваги — gated) |
| `IDEOGRAM_API_KEY` | — | «магічні» підказки (опційно) |
| `OLLAMA_API_KEY` | — | агенти (Ollama Cloud); можна ввести і в UI |
| `OLLAMA_MODEL` | `gemma4:31b` | модель агентів (розмова + зір) |
| `OLLAMA_BASE_URL` | `https://ollama.com` | адреса Ollama |
| `PORT` | `7860` | порт сервера |
